# 安装jekyll

```bash
    sudo gem install jekyll bundler
```
# 语法高亮(pygments)
[使用](http://pygments.org/)

```bash
    sudo apt-get install python-pygments
```
生成css文件  
```bash
    # -S 指定具体的样式方案
    pygmentize -f html -a .highlight -S tango > pygments.css
```
